# FastAPI

## REST API для сайта объявлений

Реализованы следующие методы для объявлений:

- создание;
- удаление;
- редактирование (обновление);
- получение по id;
- поиск по полю.

У объявления есть следующие поля:

- заголовок;
- описание;
- цена;
- автор;
- дата создания.

Авторизация и аутентификация не реализованы.
